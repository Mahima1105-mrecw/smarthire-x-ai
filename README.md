# SmartHireX

SmartHireX is an AI-powered intelligent hiring and candidate ranking platform developed for the India Runs Hackathon 2026.

## Features
- Resume Upload & Parsing
- AI Semantic Matching
- Candidate Ranking
- Skill Gap Analysis
- Recruiter Dashboard
- Explainable AI Recommendations

## Tech Stack
### Frontend
- React.js
- Tailwind CSS

### Backend
- FastAPI
- Python

### AI/NLP
- Sentence Transformers
- spaCy
- Gemini API

### Database
- MongoDB

## Folder Structure
```
SmartHireX/
│
├── frontend/
├── backend/

```

## Installation

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
